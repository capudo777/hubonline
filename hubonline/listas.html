Umaformas produtivas de organizar conteúdos.

Listas ordenadas

  1. Ondas de rádio: Varia de 3 kHz (3.000 Hz) a 300 GHz
  2. Micro ondas: Varia de 3 kHz (3.000 Hz) a 300 GHz
  3. Infravermelho: Variam entre 300 GHz e 430 THz (ou 3 x 10^11 Hz a 4,3 x 10^14 Hz)
  4. Luz visível: Aproximadamente entre 400 e 700 terahertz (THz)
  5. Ultravioleta: Variam de 8 x 10¹⁴ Hz a 3 x 10¹⁶ Hz

Listas não ordenadas

  • Azul
  • Anil
  • Vermelho
  • Laranja
  • Amarelo
  • Violeta
  • Verde
